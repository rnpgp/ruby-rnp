The `rnp` gem provides Ruby bindings to the
https://www.rnpgp.com/software/rnp/[RNP OpenPGP library] and requires RNP 0.10.0 or newer.

== Installation

Add this line to your application's Gemfile:

[source,ruby]
----
gem 'rnp'
----

[source,ruby]
----
bundle
----

Or install it yourself as:

[source,ruby]
----
gem install rnp
----
